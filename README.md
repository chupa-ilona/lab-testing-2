У межах даної роботи було реалізовано клас Cone, який моделює геометричну фігуру — конус.
Реалізований функціонал класу:

Клас містить:

конструктор з параметрами radius та height

гетери та сетери

перевизначений метод toString()

методи для обчислення основних геометричних характеристик:

volume() — обчислення об'єму конуса

surfaceArea() — обчислення повної площі поверхні

baseArea() — площа основи

getDiameter() — діаметр основи

getCircumference() — довжина кола основи

Для математичних розрахунків використано константу Math.PI.


Для перевірки коректності роботи класу було створено окремий тестовий клас ConeTests з використанням JUnit 5 (Jupiter).
